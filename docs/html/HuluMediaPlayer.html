HuluMediaPlayer

HuluMediaPlayer is the core layer for Hulu player family, it provides an unified interface for applications to handle media playback.

new HuluMediaPlayer(target)

Parameters:
Name Type Description
target HTMLVideoElement

A video element for the player to initialize.
Example
var myHuluMediaPlayer = new HuluMediaPlayer(document.querySelector("#video"));

load(options)

Load media

Parameters:
Name Type Description
options object

Initialization configs.

Properties
Name Type Description
content string | object

Metadata to build main timeline. Media source URL or MediaMetadata.

Properties
Name Type Description
src string

MediaMetadata.src Media source URL.
type string

MediaMetadata.type Media type (mp4|dash), can be resolved from the src's postfix (.mp4 and .m3u8 for mp4, .mpd for dash).
licenses object

DRM licenses.

Properties
Name Type Description
widevine string

Widevine DRM licenses URL.
playready string

PlayReady DRM licenses URL.
fairplay string

FairPlay licenses URL.
fairplayCert string

FairPlay certificate URL.
autoplay Boolean

[optional]Indicating whether playback should automatically begin as soon as enough media is available to do so without interruption, default false.
initialTime Number

[optional]The initial playback position in seconds, default 0.
maxBuffer Number

[optional] Max seconds to buffer, default 20. dash only
ADCollapsed Boolean

[optional]Indicating whether a AD period should collapse as a point (duration=0) in main timeline, default false. dash only, not ready for live
intros Array

[optional]External medias for intro. intro={type, src}, type String Intro type (preroll|postroll), url String Intro source URL. dash only
cuePoints object

[optional]Metadata to build cue points, array of cue points. non dash only
cuePointWillStart(cuePoint,done) function

[optional]Cue point is for playing additional medias without breaking the main timeline. cuePointWillStart(cuePoint) is invoked immediately before starting a cue point. It should always return a MediaMetadata (or Media source URL) array by call the done callback. If the callback never be called the player will be kept hanging. If it return something else instead of an array the CuePoint will be skipped. non dash only
Inherited From:
  • PlayerInterface#load
Overrides:
  • PlayerInterface#load
Example
var myMediaPlayer = new HuluMediaPlayer(document.querySelector("#video"));
myMediaPlayer.load({
  content: {
    type: 'dash',
    src: 'http://fiesta-xlink.test.hulu.com/dash/60240737.mpd',
  },
  cuePoints: [
    { time: 120, data: {//cue point metadata} },
    { time: 300, data: {//cue point metadata} },
    { time: 480, data: {//cue point metadata} }
  ],
  cuePointWillStart: function(cuePoint, done){
    //return media metadata for playback
    done([{ type: 'mp4', src: cuePoint.data.src}]);
  }
});
